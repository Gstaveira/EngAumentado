# EngAumentado
Projeto da disciplina Tópicos em Engenharia de Software.
# Missão 1
A missão 1 contém o link do vídeo da entrevista com a cliente, a transcrição da entrevista (feita com IA) e a elaboração do relatório de viabilidade, que analisa se a
proposta de utilização de hemogramas para vigilância epidemiológica possui viabilidade antes de dar início ao desenvolvimento do protótipo.

# Missão2
A missão 2 gerou os relatórios de planejamento e gerenciamento de projetos, análise e modelagem do sistema e o sistema em si (diretório monitoramento-hemogramas).
Para a elaboração desses entregáveis foram empregadas IA's de LLM (Gemini, GPT) para revisão dos relatórios e geração dos diagramas através da plataforma mermaidchart.com, 
GitHub Copilot para auxiliar no desenvolvimento dos códigos, junto com Webcrumbs, um assistente de IA para desenvolvimento front-end de código.
